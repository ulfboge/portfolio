# MGIS-Downloader

![MGIS-Downloader](../../assets/images/placeholder-project.png)

## Overview

A **web application developed by MundoGIS** for downloading and processing geographic data from Swedish authorities. The tool simplifies access to open Swedish geodata sources, enabling users to search, filter, and download datasets directly through a browser interface without manual navigation of authority portals.

**Developer:** MundoGIS  
**Role:** Contributor  
**Status:** Active  
**Repository:** [github.com/ulfboge/MGIS-downloader](https://github.com/ulfboge/MGIS-downloader)

---

## Supported Data Sources

- Lantmäteriet — topographic data and base maps
- Naturvårdsverket — protected areas and environmental monitoring
- SGU — geology and soil data
- Skogsstyrelsen — forest stand data
- Other Swedish open geodata authorities
